
# BI Dashboards
This document describes recommended BI dashboards for the Microloan Approval System.

## 1. Loan Performance Dashboard
**Key Visuals:**
- Loan approval rate over time
- Average loan amount
- Loan status distribution (Approved, Pending, Rejected, Closed)
- Borrower demographics

## 2. Risk Analytics Dashboard
**Visuals Include:**
- Risk level distribution (Low, Medium, High)
- Credit score heatmaps
- Risk trends over months

## 3. Payment Behavior Dashboard
**Visuals Include:**
- Monthly payments collected
- Delinquency rate
- Average repayment duration
- Payment method usage patterns

## 4. Operational Activity Dashboard
**Visuals Include:**
- User audit logs
- Blocked vs. allowed actions
- Trigger events summary
